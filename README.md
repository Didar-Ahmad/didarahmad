# Didar Ahmad Portfolio

A static portfolio site focused only on YouTube, modeling, and writing.

## Before publishing

1. Add modeling images to `images/` using `modeling-01.jpg` through `modeling-04.jpg`.
2. In `app.js`, add a YouTube Data API v3 key and channel ID. Restrict the API key to your deployed domain in Google Cloud Console.
3. Publish the repository with GitHub Pages, then add `didarahmad.com` in **Settings → Pages → Custom domain** and update your DNS records.

Do not commit an unrestricted YouTube API key to a public repository.
